Parágrafos e quebras de linhas

Você usando o comando <!>e apertando "enter". Ele coloca o código base do seu documento HTML para você. Vc pode escrever um parágrafo de qualquer jeito. Basta colocar tudo dentro do par de tags <p> e </p>

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>

Para colocar o sinal de (<) maior que e menor que (>) devemos colocar o código "& g t ;" que significa GREATER THAN para maior que e "& l t ;" que significa LESS THAN para menor que.

Para os símbolos devemos colocar o seguinte código "&'exemplo "reg" ' ; " para colocar o símbolo de marca registrada.
Para colocar valor hexadecimais devemos colocar o seguinte código " & # x 'codigo hexadecimal' ; " exemplo emojis tem valores em hexadecimais assim como as cores tmbm tem valores em hexadecimais.
